# Cálculo de ventas por semana

## Descripción
Este proyecto es un ejercicio práctico de JavaScript que permite registrar las ventas de 2 semanas, considerando 3 días por semana.  
A partir de los datos ingresados por el usuario, el programa calcula el total de ventas y el promedio de las 2 semanas.

## Objetivo
Practicar el uso de:
- Variables.
- Estructuras repetitivas `for` y `do...while`.
- Ingreso de datos con `prompt()`.
- Mostrar resultados en pantalla con Bootstrap.

## Funcionalidades
- Solicita las ventas de 2 semanas.
- Registra 3 días por cada semana.
- Calcula el total general de ventas.
- Calcula el promedio de ventas de las 2 semanas.
- Muestra el resultado dentro de un recuadro con Bootstrap.

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- Bootstrap 5

## Estructura del proyecto
- `index.html`: contiene la estructura principal de la página.
- `app.js`: contiene la lógica del ejercicio.

## Cómo ejecutar el proyecto
1. Clona o descarga el repositorio.
2. Abre el archivo `index.html` en tu navegador.
3. Ingresa las ventas solicitadas en los cuadros de diálogo.
4. Visualiza el reporte final en pantalla.

## Aprendizajes
Con este ejercicio se refuerza:
- El uso de ciclos anidados.
- La validación básica de datos.
- El trabajo con arreglos.
- La construcción de resultados dinámicos en el DOM.
